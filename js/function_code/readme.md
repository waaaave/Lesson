- 纯函数的概念
  一个函数 用相同的参数传入， 返回值是一样的话，这就叫做纯函数
  还要不产生副作用
  reducer
  initialState + action = new state

- 类式组件的缺点和函数是组件的优点
  1. 类式编程风格，从编程思想上就决定了很复杂 + redux 就更复杂
  2. 函数式编程风格([].reducer) 
    完成和类式编程一样的功能，同时更优雅 更简洁
  3. 钩子函数额引入， use*** 不要轻易打扰函数的存粹功能( 显示UI ) 
    use 使用一个hook 内置的或者自定义的
    完成了函数式编程 纯函数得以体现
